# Rental House Management System  
A web-based system developed for ISP550 to help users register, log in, upload rental applications, and submit supporting documents such as income proof and identification. The system allows an admin or owner to review submissions and manage rental applications.

---

## Features

### User Features
- User Registration  
- User Login / Logout  
- Submit Rental Application  
- Upload Supporting Documents:
  - Income Proof (PDF/JPG/PNG)
  - IC/Passport Copy
  - Additional Supporting Documents

### Rental Application Features
- Application form (details, income, attachments)  
- File upload handling  
- View application status  

### Authentication Features
- Secure login using PHP + MySQL  
- Password hashing (if implemented)  
- User session management  

---

## Technologies Used
- **PHP** – Backend logic  
- **HTML, CSS, JavaScript** – Frontend  
- **MySQL** – Database  
- **XAMPP** – Local server environment
